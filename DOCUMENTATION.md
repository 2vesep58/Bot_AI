        └── utils/              # Утилиты

## Версионирование документации

Документация обновляется вместе с кодом. Каждый файл содержит информацию о версии в комментариях.

## Обновление документации

При добавлении новой функции:
1. Обновите соответствующий файл документации
2. Обновите README.md
3. Обновите CHANGES.md
4. Обновите FAQ.md (если необходимо)

## Форматирование документации

Все файлы документации используют Markdown формат.

### Основные элементы

- **Заголовки:** # H1, ## H2, ### H3
- **Жирный текст:** **текст**
- **Курсив:** *текст*
- **Код:** `код`
- **Блоки кода:** ```язык ... ```
- **Списки:** - элемент или 1. элемент
- **Ссылки:** [текст](url)
