# Команды бота

## Основные команды

### /start
**Описание:** Запуск бота и приветственное сообщение

**Использование:**
```
/start
```

**Ответ:**
```
Привет! Я эхо-бот. Отправь мне любое сообщение, и я его повторю.
```

**Когда использовать:** При первом запуске бота

---

### /help
**Описание:** Справка по доступным командам

**Использование:**
```
/help
```

**Ответ:**
```
Доступные команды:
/start - запуск бота
/help - справка по командам
/chatgpt - начало диалога с ИИ
/stop - завершение диалога с ИИ
```

**Когда использовать:** Когда вам нужна справка по командам

---

### /chatgpt
**Описание:** Активирует режим взаимодействия с ИИ Amvera (модель gpt-5)

**Использование:**
```
/chatgpt
```

**Ответ:**
```
Отправьте мне сообщение, и я отвечу с помощью ИИ. Чтобы выйти из режима ИИ, отправьте /stop.
```

**Когда использовать:** Когда вы хотите общаться с ИИ

**Примеры:**
```
/chatgpt
Что такое Python?

⏳ Обрабатываю ваш запрос...
Python - это высокоуровневый язык программирования, известный своей простотой и читаемостью...
```

---

### /stop
**Описание:** Деактивирует режим взаимодействия с ИИ и возвращает бота в режим эхо

**Использование:**
```
/stop
```

**Ответ:**
```
Вы вышли из режима общения с ИИ. Теперь я буду работать как эхо-бот.
```

**Когда использовать:** Когда вы хотите выйти из режима ИИ

---

## Режимы работы

### Режим эхо (по умолчанию)
**Описание:** Бот повторяет все сообщения пользователя

**Использование:**
```
Привет!
```

**Ответ:**
```
Привет!
```

**Когда использовать:** Когда вы не активировали режим ИИ

---

### Режим ИИ
**Описание:** Бот отвечает с помощью ИИ Amvera

**Активация:**
```
/chatgpt
```

**Использование:**
```
Что такое машинное обучение?
```

**Ответ:**
```
⏳ Обрабатываю ваш запрос...
Машинное обучение - это раздел искусственного интеллекта, который позволяет компьютерам учиться на основе данных...
```

**Деактивация:**
```
/stop
```

## Обработка ошибок

### Ошибка: Timeout при обращении к ИИ

**Сообщение:**
```
Произошла ошибка при обращении к ИИ. Пожалуйста, попробуйте позже.
```

**Причины:**
- Медленное интернет соединение
- Перегруженность API Amvera
- Сложность запроса

**Решение:**
- Попробуйте позже
- Проверьте интернет соединение
- Упростите запрос

### Ошибка: Неправильный токен

**Сообщение:**
```
Произошла ошибка при обращении к ИИ. Пожалуйста, попробуйте позже.
```

**Причины:**
- Неправильный токен Amvera LLM
- Истекший токен

**Решение:**
- Проверьте токен в файле `.env`
- Получите новый токен на сайте Amvera

## Советы по использованию

### 1. Используйте четкие и конкретные вопросы

**Хорошо:**
```
Что такое Python и для чего он используется?
```

**Плохо:**
```
Расскажи про язык
```

### 2. Используйте контекст

**Хорошо:**
```
Я изучаю Python. Как написать функцию для вычисления факториала?
```

**Плохо:**
```
Как написать функцию?
```

### 3. Разбивайте сложные вопросы на части

**Хорошо:**
```
Что такое асинхронное программирование?
Как использовать async/await в Python?
```

**Плохо:**
```
Расскажи мне все об асинхронном программировании в Python
```

### 4. Используйте команду /stop для выхода

**Хорошо:**
```
/stop
```

**Плохо:**
```
Выход
Выйти
Стоп
```

## Ограничения

### Длина сообщения
- Максимальная длина сообщения: 4096 символов (ограничение Telegram)

### Время ответа
- Максимальное время ответа: 30 секунд

### Количество запросов
- Ограничения зависят от плана Amvera LLM

### Языки
- Поддерживаются все языки, которые поддерживает Amvera LLM
