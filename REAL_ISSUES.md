# Реальные проблемы

## Проблема 1: Ошибка 402 "Run out of tokens"

**Что это значит**: Это НЕ ошибка токена авторизации. Это ошибка **биллинга на Amvera**.

**Сообщение об ошибке**:
```
{"message":"Failed to get access from billing for operation. Run out of tokens."}
```

**Причины**:
1. Недостаточно средств на аккаунте Amvera
2. Исчерпана квота на использование API
3. Лимит запросов превышен

**Решение**:
1. Зайти в личный кабинет Amvera
2. Проверить баланс/квоту
3. Пополнить баланс или увеличить лимит
4. Проверить, не установлены ли ограничения на использование API

## Проблема 2: Конфликт двух экземпляров бота

**Ошибка в логах**:
```
TelegramConflictError: Telegram server says - Conflict: terminated by other getUpdates request; 
make sure that only one bot instance is running
```

**Причина**: 
- Локальный бот работает на компьютере
- Amvera контейнер тоже работает одновременно
- Оба пытаются получать обновления от Telegram для одного и того же бота

**Решение**:
1. Остановить локальный бот
2. ИЛИ остановить Amvera контейнер
3. Работать только с одним экземпляром

## Текущее состояние

**На Amvera**: Контейнер тоже работает

**Нужно выбрать**:

- Либо работать на Amvera (остановить локальный бот)

## Как проверить баланс Amvera

1. Зайти на https://amvera.ru
2. Перейти в личный кабинет
3. Проверить раздел "Биллинг" или "Баланс"
4. Проверить квоту на использование API

## Как остановить локальный бот

```powershell
# Найти процесс
Get-Process python | Where-Object {$_.Id -eq 9472}

# Остановить
Stop-Process -Id 9472 -Force
```

## Как остановить Amvera контейнер

1. Зайти в панель управления Amvera
2. Найти приложение "amvera-delana-run-test-bot"
3. Нажать "Остановить" или "Disable"
